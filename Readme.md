﻿# Overview

The ***Linq.Next*** project enhances the linq and the collections with an extra features.

- Linq:
  - CompareTo
  - Split
  - SplitBefore
  - SplitAfter
  - TagFirst
  - TagLast
  - TagFirstLast
  - WithPrev
  - WithNext
  - WithPrevNext

- Collections:
  - StatefulEnumerator
  - PeekableEnumerator

- System
  - Option
